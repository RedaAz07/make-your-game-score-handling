# 🎮 Brick Bricker Game

Welcome to **Brick Bricker** — a classic arcade-style game made with HTML, CSS, and JavaScript.

## 🧱 How to Play

- Move your paddle using your mouse or keyboard.
- Keep the ball from falling.
- Break all the bricks to win!
